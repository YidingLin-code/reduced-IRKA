# reduced-IRKA
MATLAB codes for 'A reduced-IRKA method for large-scale H2-optimal model order reduction'

This repository contains MATLAB codes of R-IRKA and TR-IRKA for H2 optimal model order reduction.


If you use this code, please cite the following paper.

Yiding Lin and Valeria Simoncini, A reduced-IRKA method for large-scale $\mathcal{H}_2$-optimal model order reduction, Journal of Scientific Computing, 2026
